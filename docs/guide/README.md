# 介绍

本组件库基于Vue，主要用于构建大屏（**全屏**）数据展示页面，具有多种类型组件可供使用
* **边框**

带有不同边框的盒子

* **装饰**

用来点缀页面效果，增加视觉效果

* **图表**

图表类组件类似于eCharts 但更为轻量精简，资源消耗也更小

* **其他**

## 使用

首先你需要下载它
[下载地址](http://datav.jiaminghi.com)

然后将下载后的压缩包进行解压，放入你的项目中并在main.js中进行引用

```js
// main.js
import dataV from './some/dir/datav'

Vue.use(dataV)
```

完成上述步骤后即可在你的项目中自由的使用所有的组件了
